# 系统分析报告 - 智学助手 (ZhixueHelper)

## 项目概述

智学助手是一个由大语言模型(LLM)驱动的个性化学习管理系统，提供知识库管理、智能练习/考试生成和学习分析功能。

## 技术栈

| 类别 | 技术 |
|------|------|
| 前端框架 | Vue 3 (^3.2.13) |
| 构建工具 | Vue CLI 5 |
| 状态管理 | Pinia (^3.0.4), Vuex 4 |
| 路由 | Vue Router 4 (^4.1.6) |
| UI库 | Element Plus (^2.13.5) |
| HTTP客户端 | Axios (^1.13.6) |
| 图表 | ECharts (^6.0.0), D3.js (^7.9.0) |

## 主要功能模块

### 1. 认证系统 (`/login`, `/register`, `/forgot-password`)
- 用户注册（邮箱验证）
- 登录（用户名/邮箱 + 密码）
- 密码找回

### 2. 知识库管理 (`/knowledge`, `/knowledge-graph`)
- 文档上传（PDF、文本、图片）
- 文档列表（标签和关键词筛选）
- 知识图谱可视化
- 文档搜索和检索

### 3. 题目/考试生成 (`/exam`)
- 基于知识点的AI试卷生成
- 可配置难度（简单/中等/困难）
- 支持多种题型（单选、多选、填空、问答）

### 4. 练习与考试 (`/practice`)
- 练习模式（即时反馈）
- 考试模式（计时评估）
- 答案提交和评分

### 5. 错题本 (`/wrong-questions`)
- 自动追踪错题
- 薄弱环节强化练习

### 6. 学习分析 (`/analytics`)
- 学习统计仪表板
- 知识点掌握雷达图
- 学习趋势可视化
- AI学习建议

### 7. 用户中心 (`/profile`, `/settings`)
- 个人信息管理
- 头像定制
- 密码修改

## API集成

前端通过 Bearer token 认证与后端 API (`http://localhost:8000/api`) 通信。
