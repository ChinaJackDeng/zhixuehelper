# 题目生成前端对接完整指南

## 📋 系统架构

```
┌─────────────────────────────────────────────────────────────────┐
│                          前端 (QuestionGenerator.vue)           │
└────────────────────────┬────────────────────────────────────────┘
                         │ 用户操作
                         ↓
                    ┌────────────────┐
                    │ 发起生成任务   │ POST /generate
                    │ (返回 task_id) │
                    └────────┬───────┘
                             │
        ┌────────────────────┴────────────────────┐
        │ 开启双轨并行查询                        │
        ↓                                         ↓
   ┌──────────────┐                    ┌──────────────────┐
   │ 轨道 A       │ 每 3 秒             │ 轨道 B          │
   │ 查询进度    │ 轮询                │ 加载题目        │
   │ progress    │ /generate/{id}      │ /preview/{id}   │
   └────┬─────────┘                    └────┬────────────┘
        │                                   │
        │ status='ready'                     │ 加载生成的题目
        ↓                                   ↓
   ┌──────────────────────────────────────────────┐
   │ 题目预览显示在右侧卡片中                     │
   │ 用户可以翻页查看所有题目                     │
   └──────────┬───────────────────────────────────┘
              │
              │ 用户点"保存题目"
              ↓
         ┌────────────────┐
         │ POST /confirm  │ 保存到数据库
         │ Redis→Database │
         └────────────────┘
              │
              ↓
         ✅ 保存成功
```

## 🔑 关键概念

### 两阶段存储模式

| 阶段 | 存储位置 | 何时使用 | 接口 |
|------|--------|--------|------|
| **生成阶段** | Redis（临时） | 题目生成完成到用户确认前 | `/preview/{id}` ⭐ |
| **保存阶段** | 数据库（永久） | 用户点"保存题目"后 | `/task/{id}` |

**重要**：不能混淆这两个接口，否则前端看不到题目！

### 2️⃣ 轨道 A：轮询进度

```javascript
// 每 3 秒查询一次生成进度
const startProgressPolling = (taskId) => {
  const poll = async () => {
    const progress = await getGenerationProgress(taskId)
    
    // 更新显示
    generationProgress.value = progress.progress  // 0-100
    currentStep.value = progress.current_step     // "正在生成多选题..."
    
    if (progress.status === 'ready') {
      // 生成完成，停止轮询
      generating.value = false
      return
    }
    
    if (progress.status === 'failed') {
      ElMessage.error('生成失败')
      return
    }
    
    // 继续轮询
    setTimeout(poll, 3000)
  }
  
  poll()
}
```

**进度响应示例**:
```json
{
  "task_id": "abc-def-123",
  "status": "generating",
  "progress": 45,
  "current_step": "正在生成第 2/2 批题目（3 题）",
  "total_questions": 8,
  "generated_questions": 4,
  "error_message": null
}
```

**状态取值**:
- `pending` - 等待处理
- `generating` - 正在生成中
- `ready` - 生成完成，可以预览和保存
- `failed` - 生成失败

### 3️⃣ 轨道 B：加载题目（关键！）

```javascript
// 立即开始加载题目，不要等待轨道 A 完成
const loadQuestions = async (taskId, page = 1) => {
  try {
    // ⭐ 使用预览接口从 Redis 获取临时题目
    const result = await getQuestionPreview(taskId, {
      page: page,
      page_size: 10
    })
    
    console.log('预览数据:', result)
    // {
    //   "total": 8,
    //   "page": 1,
    //   "page_size": 10,
    //   "data": [题目数组],
    //   "status": "ready",
    //   "task_type_distribution": { "single": 5, "multi": 3 }
    // }
    
    // 转换数据格式
    const questions = result.data.map(q => ({
      type: q.type,
      content: q.content,
      stem: q.content,  // 题干
      options: Object.entries(q.options || {}).map(([k, v]) => ({
        key: k,
        text: v
      })),
      answer: q.answer,
      analysis: q.analysis,
      difficulty: q.difficulty
    }))
    
    // 显示题目列表
    generatedQuestions.value = questions
    
  } catch (error) {
    console.error('加载题目失败:', error)
  }
}

// 在轴道 A 中调用它
startDualTracking(taskId) {
  startProgressPolling(taskId)        // 轨道 A
  loadQuestions(taskId, 1, 10)         // 轨道 B
}
```

### 5️⃣ 保存题目（确认）

const saveQuestions = async () => {

## 🎯 核心实现代码

### 完整的双轨加载实现

```javascript
// 1. 创建生成任务
const generateQuestions = async () => {
  const response = await apiGenerateQuestions({
    question_types: configForm.questionTypes,
    document_ids: configForm.selectedFiles,
    tags: configForm.tags,
    difficulty: configForm.difficulty,
    question_count: totalCount
  })
  
  currentTaskId.value = response.task_id
  generating.value = true
  
  // ⭐ 启动双轨查询
  startDualTracking(response.task_id)
}

// 2. 轨道 A：轮询进度
const startProgressPolling = (taskId) => {
  const poll = async () => {
    try {
      const progress = await getGenerationProgress(taskId)
      
      // 更新进度
      generationProgress.value = progress.progress
      currentStep.value = progress.current_step
      totalGenerated.value = progress.generated_questions
      
      // 判断状态
      if (progress.status === 'ready') {
        generating.value = false
        return
      }
      
      if (progress.status === 'failed') {
        ElMessage.error('生成失败')
        return
      }
      
      // 继续轮询
      pollTimer.value = setTimeout(poll, 3000)
      
    } catch (error) {
      console.error('轮询失败:', error)
    }
  }
  
  poll()
}

// 3. 轨道 B：加载题目
const loadQuestions = async (taskId, page = 1) => {
  try {
    // ⭐⭐⭐ 关键：使用 getQuestionPreview 而不是 getTaskQuestions
    const result = await getQuestionPreview(taskId, {
      page: page,
      page_size: 10
    })
    
    if (result.data && result.data.length > 0) {
      const questions = result.data.map(q => ({
        ...q,
        type: q.type === 'choice' ? 'single' : q.type,
        stem: q.content,
        options: q.options ? Object.entries(q.options).map(([k, v]) => ({
          key: k,
          text: v
        })) : [],
        difficulty: q.difficulty || 'medium'
      }))
      
      if (page === 1) {
        generatedQuestions.value = questions
        currentQuestionIndex.value = 0
      } else {
        generatedQuestions.value.push(...questions)
      }
      
      totalGenerated.value = result.total
    }
    
  } catch (error) {
    console.error('加载题目失败:', error)
  }
}

// 4. 启动双轨
const startDualTracking = (taskId) => {
  startProgressPolling(taskId)
  loadQuestions(taskId, 1)
}

// 5. 保存题目
const saveQuestions = async () => {
  try {
    const response = await confirmSaveQuestions(currentTaskId.value)
    ElMessage.success(`保存成功：${response.saved_count} 题`)
    resetGenerator()
  } catch (error) {
    ElMessage.error('保存失败')
  }
}
```

## ⚠️ 常见错误

### ❌ 错误 1：使用错误的接口

```javascript
// ❌ 错误 - 查询数据库（题目还没保存）
const result = await getTaskQuestions(taskId)

// ✅ 正确 - 查询 Redis（临时题目）
const result = await getQuestionPreview(taskId)
```

### ❌ 错误 2：等待进度完成才加载题目

```javascript
// ❌ 错误 - 串行流程，慢
const progress = await getGenerationProgress(taskId)
if (progress.status === 'ready') {
  const questions = await getQuestionPreview(taskId)  // 等待太久
}

// ✅ 正确 - 并行流程，快
startProgressPolling(taskId)  // 持续轮询进度
loadQuestions(taskId)         // 同时立即加载题目
```

### ❌ 错误 3：只轮询一次进度

```javascript
// ❌ 错误 - 只查询一次
const progress = await getGenerationProgress(taskId)

// ✅ 正确 - 持续轮询
const poll = async () => {
  const progress = await getGenerationProgress(taskId)
  if (progress.status !== 'ready') {
    setTimeout(poll, 3000)  // 3 秒后再查
  }
}
poll()
```

## 📊 API 参考

### getGenerationProgress(taskId)

**用途**：查询题目生成进度

**请求**:
```javascript
GET /api/exam/questions/generate/{taskId}
```

**响应**:
```json
{
  "task_id": "string",
  "status": "pending|generating|ready|failed",
  "progress": 0-100,
  "current_step": "string",
  "total_questions": number,
  "generated_questions": number,
  "error_message": "string or null"
}
```

---

### getQuestionPreview(taskId, params)

**用途**：预览临时生成的题目（从 Redis 读取）

**请求**:
```javascript
GET /api/exam/questions/preview/{taskId}?page=1&page_size=10
```

**响应**:
```json
{
  "total": number,
  "page": number,
  "page_size": number,
  "data": [
    {
      "type": "single|multi|judge|fill|essay",
      "content": "题目内容",
      "options": { "A": "选项A", "B": "选项B", ... },
      "answer": "A",
      "analysis": "解析",
      "difficulty": "easy|medium|hard",
      "score": number,
      "tags": []
    }
  ],
  "status": "ready|generating|failed",
  "task_type_distribution": { "single": 5, "multi": 3 }
}
```

### confirmSaveQuestions(taskId)

**响应**:
```json
{

```
