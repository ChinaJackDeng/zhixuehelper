## 📝 注意事项

### 1. API 地址配置
确保前端 API 地址正确：
```javascript
// src/utils/request.js
baseURL: 'http://127.0.0.1:8000/api'  // 根据你的后端地址修改
```

### 2. CORS 配置
确保后端已配置跨域：
```python
# FastAPI 示例
app.add_middleware(
    CORSMiddleware,
    allow_origins=["http://localhost:8080"],
    allow_credentials=True,
    allow_methods=["*"],
    allow_headers=["*"],
)
```

### 3. Token 认证
确保登录状态正常，token 有效：
```javascript
// localStorage 中应该有 access_token
localStorage.getItem('access_token')
```

### 4. 错误处理
查看浏览器控制台和后端日志，及时处理错误。
